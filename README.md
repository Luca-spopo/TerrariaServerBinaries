# TerrariaServerBinaries

This repo is used only to upload the binaries of the MobileTerrariaServer. It's supposed to be updated when an update is pushed on the vanilla server.

LinuxMobileServer.zip can be downloaded from https://github.com/Luca-spopo/TerrariaServerBinaries/releases/download/vanilla-mobile-1405/LinuxMobileServer.zip
(Replace 1405 with any version)

Unzipping LinuxMobileServer.zip yields a folder "ServerLinux" which contains the server binary
